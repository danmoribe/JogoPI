# Block Blaster Web

Clonagem do famoso jogo de puzzle Block Blaster, desenvolvida para Web (Desktop e Mobile Friendly).

## Como Jogar
1. Arraste as peças da parte inferior para o tabuleiro 8x8.
2. Complete linhas horizontais ou colunas verticais para eliminá-las.
3. Ganhe pontos por cada bloco colocado e bônus por linhas limpas.
4. O jogo termina quando não houver mais espaço para as peças disponíveis.

## Tecnologias Utilizadas
- HTML5 & CSS3
- JavaScript (ES6+ com Módulos)
- LocalStorage para Recorde

## Como Executar

### Opção 1: Abrir no Navegador (Simples)
Basta clicar duas vezes no arquivo `index.html`. O jogo agora está configurado para funcionar diretamente pelo seu explorador de arquivos sem a necessidade de um servidor.

### Opção 2: VS Code (Live Server)
Se estiver usando o VS Code, instale a extensão **Live Server** e clique em "Go Live".

### Opção 2: NodeJS
Se tiver o NodeJS instalado, execute:
```bash
npx serve .
```

### Opção 3: Abrir no Navegador
Em alguns navegadores modernos, você pode abrir o `index.html` diretamente, mas recomendamos o uso de um servidor para garantir que todos os recursos funcionem corretamente.
